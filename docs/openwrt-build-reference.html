01

Порядок выполнения пайплайна

Каждый запуск workflow состоит из двух jobs: setup (формирует матрицу устройств) и build (собирает прошивку по матрице). Ниже — все шаги job build в порядке выполнения.

01
Free up disk space
ubuntu-22.04 runner
● Всегда

Удаляет Android SDK, .NET, Haskell, Ghc, Swift, Chromium, PowerShell, Docker-образы, браузеры, MySQL, PHP. Освобождает ~15–25 ГБ для сборки.

02
Checkout
actions/checkout@main
● Всегда

Клонирует текущий репозиторий с sh-скриптами, конфигами устройств, files/, patches/.

chmod 755 на files/usr/bin/* и files/etc/uci-defaults/*
● Всегда

⚠Условие: download_binaries_agh = true
⚠+ вариант НЕ switch, НЕ clear, НЕ crystal_clear

Download Sing-Box binary
linux-arm64 tar.gz → files/usr/bin/sing-box
⚠Условие: download_binaries_sb ≠ 'no'

latest → /releases/latest | 1.12.x → фильтр по тегу v1.12.* (только stable, не pre-release)
06
Configure Build Variables
Определяет REPO_URL, REPO_BRANCH, TARGET_RELEASE, AUTO_COMMIT_HASH, + переменные устройства
● Всегда
Логика ветки:
• master → для OpenWrt меняется на main; TARGET_RELEASE = snapshots
• Стабильные ветки: ищет последний тег через GitHub API (напр. v25.12.*), берёт его SHA → AUTO_COMMIT_HASH; TARGET_RELEASE = releases/25.12.1
• Если указан commit_hash вручную — авто-поиск пропускается
• Подключает конфиг устройства из ./config/conf/${{ matrix.target }}

Описание вариантов

standard базовый

Полный набор из WRT.config. HomeProxy + sing-box full + AGH + youtubeUnblock + SQM + BBR. Рекомендован для роутеров с 1+ ГБ RAM.

minimal −SQM

Standard без SQM (sqm, sqm-scripts, luci-app-sqm, luci-i18n-sqm-ru). Экономит место и время сборки при отказе от активного управления очередями.

clear без прокси и BBR

Убирает весь прокси-стек, AGH, youtubeUnblock, avahi. Отключает BBR. Чистый роутер без DPI-обхода. Бинарники не скачиваются. SQM — остаётся (кроме BBR).

crystal_clear ультра-минимум

Агрессивная зачистка: всё из clear + coreutils, findutils, USB-хранилище, crypto kmod, VPN kmod, QoS, ipset, macvlan, множество библиотек. Удаляет zz1-final-offline-setup.sh и speedtest.tar.gz. Без BBR.

switch L2-коммутатор

Тот же набор удалений что и crystal_clear, но zz1-final-offline-setup.sh сохраняется и активирует режим коммутатора: bridge всех портов, DHCP-клиент, без DHCP-сервера, аппаратный offloading.

podkop +podkop

Minimal (без SQM) + podkop + luci-app-podkop + sing-box. HomeProxy удаляется из конфига. На slateax/ax59u sing-box заменяется на sing-box-tiny.

passwall passwall2 + xray

HomeProxy и sing-box удаляются. Добавляются: passwall2-фид, luci-app-passwall2 (nftables, xray, v2ray-plugin, IPv6 NAT), xray-core, geoview, v2ray-geodata. Все SS-компоненты отключены явно. SQM не трогается.

v2raya v2rayA

HomeProxy и sing-box удаляются. Добавляются: xray-core, v2rayA, v2ray-geodata, luci-app-v2raya, luci-i18n-v2raya-ru.

xray luci-app-xray

HomeProxy и sing-box удаляются. Добавляются: luci-app-xray (из фида yichya), xray-core, v2ray-geodata, geoview.

WRT-part2.sh — Фиксы версий

DIY_P2_SH Запускается после feeds update, до feeds install. Исправляет Makefile-ы пакетов в фидах.

ДействиеУсловиеДетали
Обновить youtubeUnblock до latest main НЕ (clear / crystal_clear / switch)
  1. Находит Makefile в feeds/youtubeUnblock/youtubeUnblock/
  2. Получает SHA последнего коммита через GitHub API (fallback: git ls-remote)
  3. Обновляет PKG_REV:= или PKG_SOURCE_VERSION:=
  4. Получает реальный VERSION из upstream Makefile, формирует PKG_VERSION:=X.Y.Z.YYYYMMDD
  5. Сбрасывает PKG_RELEASE:=1
  6. Удаляет PKG_HASH, заменяет PKG_MIRROR_HASH:=skip
Фикс Rust 1.90.0 ЗАКОММЕНТИРОВАНО Был: download-ci-llvm=true → if-unchanged. Не нужен в текущей версии.
Фикс Python PGO ЗАКОММЕНТИРОВАНО Только для ветки master/main: отключал PGO и тесты. Ускорял сборку.
Фикс ARM64_CONTPTE (nanopi-r5s + master) ЗАКОММЕНТИРОВАНО Добавлял CONFIG_ARM64_CONTPTE=y в конфиг ядра Rockchip чтобы избежать интерактивного запроса при сборке.

zz1-final-offline-setup.sh — Первый запуск роутера

/etc/uci-defaults/zz1-final-offline-setup.sh Выполняется автоматически один раз при первой загрузке через механизм uci-defaults. После успешного выполнения (exit 0) удаляется системой. Создаёт lock-файл /root/.setup_completed. Перезагружает роутер через 120 секунд.

Блок настройки Условие Что делает
Определение модели Всегда Читает ubus call system board → определяет ROUTER_NAME (oWRT/iWRT), ROUTER_MODEL_NAME (Brume2/R5S/R6S/SLATEX/RB5009), HOSTNAME_PATTERN.
Чтение варианта Всегда cat /etc/build_variant → CURRENT_VARIANT. Создан в WRT-part3.sh.
Удаление FullCone NAT из firewall Всегда Удаляет option fullcone '1' из /etc/config/firewall (будет восстановлен позже если модуль доступен).
Часовой пояс Всегда uci set system.@system[0].zonename='Europe/Moscow'
Определение пакетного менеджера Всегда apk → /etc/apk/; opkg → /etc/opkg/. Для APK: проверяет архитектуру, копирует ключи подписи из /root/apps/.
Очистка distfeeds Всегда Оставляет только строки с: targets, base, luci, packages, routing, telephony, video. Создаёт бэкап.
Авто-добавление kmods репозитория Только APK Определяет версию ядра через apk list -I kernel, формирует URL kmods и добавляет в distfeeds.list.
Установка sing-box из /root/apps/ Если файл существует Распаковывает sing-box.tar.gz → /usr/bin/sing-box, chmod +x.
Установка speedtest из /root/apps/ Если файл существует Распаковывает speedtest.tar.gz → /usr/bin/speedtest, chmod +x.
Настройка HomeProxy Если /etc/init.d/homeproxy существует Патчит firewall_post.ut (отключает dns_hijacked).
Создаёт update_firewall_rules.sh: условно добавляет/удаляет правила firewall в зависимости от режима (tun/server).
Патчит init.d/homeproxy: вызывает скрипт-помощник при start/stop.
Включает службу.
Настройка Passwall2 variant = passwall + lock отсутствует Отключает DNS redirect (используем AGH), устанавливает remote_dns=127.0.0.1:53, включает службу.
Настройка youtubeUnblock Всегда (если пакет установлен) Создаёт пустые ipset-ы, если их нет: dpi_ips, no_dpi_ips, dpi_guest_ips.
Патчит ruleset.uc: добавляет ct-packets threshold для offloading.
Пишет /usr/share/nftables.d/ruleset-post/537-youtubeUnblock.nft с правилами postrouting (queue 537, bypass).
Настройка internet-detector Если init.d существует Отключает автозапуск, меняет START=99.
Настройка owut (attendedsysupgrade) Всегда OpenWrt → https://sysupgrade.openwrt.org. ImmortalWrt → https://asu-2.kyarucloud.moe.
Настройка AdGuardHome Если /usr/bin/AdGuardHome и конфиг существуют Устанавливает workdir /opt/AdGuardHome, user/group root.
Патчит init-скрипт (jail-режим или классический): добавляет logging в файл.
Создаёт LuCI-меню в admin > network > adguardhome (перенаправление на порт 8080).
Переводит dnsmasq на порт 54.
Запускает службу.
Патч SQM Если /usr/lib/sqm/run.sh существует Создаёт /opt/sqm_custom.conf с diffserv4 nowash. Патчит run.sh для загрузки кастомных опций. Перезапускает SQM если он был включён.
CSS патч LuCI Bootstrap Всегда Добавляет в /www/luci-static/bootstrap/cascade.css: 100% ширина на мобильных (<1200px или touch), 50% ширина на десктопе.
FullCone NAT Если модуль nft_fullcone.ko доступен uci set firewall.@defaults[0].fullcone='1'
TCP BBR Если tcp_bbr.ko доступен Записывает в /etc/sysctl.conf: net.core.default_qdisc = fq_codel + net.ipv4.tcp_congestion_control = bbr.
Отключение phy-leds Если init.d существует /etc/init.d/phy-leds disable
Режим коммутатора (switch) variant = switch + lock отсутствует Удаляет network.lan, wan, wan6.
Автоматически определяет физические порты (исключает lo, wlan, br-, tun, sit…).
При DSA-архитектуре: исключает eth0.
Создаёт br-lan (bridge всех портов, STP=1, IGMP snooping=1).
LAN = DHCP-клиент, IPv6 отключён.
Отключает DHCP-сервер и odhcpd.
Firewall: удаляет WAN-зону и forwarding, включает аппаратный offloading.
Удаляет все правила firewall и dhcp.wan.
Синхронизация времени Всегда hwclock -s (читает RTC). При наличии интернета: ntpd -q -n -p ru.pool.ntp.org, затем hwclock -w.
Обновление баннера Всегда WIRELESS → NETWORK. Добавляет: Kernel Version: X.Y.Z, Build Variant: standard (2026-01-01).
Обновление hostname Всегда Формат: ROUTER_MODEL_NAME-ROUTER_NAME (напр. RB5009-oWRT, R6S-iWRT). Для switch: RB5009-switch.
Перевод «File Manager» Всегда Меняет в JSON меню на «Файловый менеджер».
Перезагрузка Всегда Через 120 секунд в фоне: (sleep 120; sync; reboot) &

Режимы combo / all — матрица сборки

При выборе target = combo или target = all вместо одного устройства запускается матрица из нескольких параллельных build-задач. Вариант из параметра при этом игнорируется — матрица жёстко задана в коде (combo) или применяется к ограниченному набору устройств (all).

targetУстройствоВариантПояснение
combo nanopi-r6s minimal Жёстко захардкожено в job setup. Параметр variant игнорируется.
rb5009 clear
rb5009 switch
all nanopi-r6s + rb5009 из параметра variant Запускает выбранный вариант для обоих устройств. Остальные устройства (r5s, slateax, ax59u, brume2) в all не включены.
конкретное устройство одно из 6 из параметра variant Один билд.
Стратегия fail-fast: отключена (fail-fast: false). Если один билд в матрице падает — остальные продолжают выполняться.

Имена артефактов

Формат: {source}_{branch}_{variant}_{target}_{YYYYMMHHMM}

ПримерОписание
